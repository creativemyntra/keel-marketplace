# Keel Marketplace

Claude Code plugin marketplace for [Keel](https://github.com/creativemyntra/keel).

## Add this marketplace

```
/plugin marketplace add creativemyntra/keel-marketplace
```

## Install Keel

```
/plugin install keel@keel-marketplace
```

Then, one time in your repo:

```bash
bash setup.sh
```

## What Keel is

Governed AI delivery for any repository — 12 specialist agents, enforced handoffs,
evidence-cited memory, destructive-command blocking, token-saving codebase queries.
